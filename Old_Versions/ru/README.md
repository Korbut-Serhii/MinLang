# MinLang — v0.9 документация

Минималистичный язык программирования. Максимум возможностей, минимум символов.

---

## Установка и запуск

### Вариант 1 — через Python (для разработки)
```bash
python3 minlang.py program.minl     # запустить файл
python3 minlang.py                # интерактивный REPL
```

### Вариант 2 — команда `minlang` в терминале (Windows)
Создай файл `minlang.bat` и положи его в папку, которая есть в PATH
(например `C:\Windows\System32` или создай `C:\tools\` и добавь в PATH):
```bat
@echo off
python3 "C:\полный\путь\к\minlang.py" %*
```
Теперь можно писать просто: `minlang program.minl`

### Вариант 3 — .exe без Python (для дистрибуции)
```bash
pip install pyinstaller
pyinstaller --onefile minlang.py
```
В папке `dist/` появится `minlang.exe` — один файл, Python внутри,
ничего устанавливать не нужно. Положи его в PATH — готово.

## Переменные

| Синтаксис | Что делает |
|---|---|
| `L x = 5` | Объявить новую переменную x |
| `x = 10` | Переприсвоить существующую |
| `x[0] = 99` | Изменить элемент списка |

Типы данных:
```
L n   = 42          ## целое число (int)
L f   = 3.14        ## дробное (float)
L s   = "привет"    ## строка
L b   = T           ## булево: T = true, F = false
L nil = nil         ## null / None
L lst = [1, 2, 3]   ## список
L dct = {"key": 1}  ## словарь
```

## Вывод

| Синтаксис | Что делает |
|---|---|
| `pt "текст"` | Вывести без переноса строки |
| `ptl "текст"` | Вывести с переносом строки |
| `pt f"x = {x}"` | F-строка — подставляет переменные |

```
L name = "Мир"
ptl f"Привет, {name}!"     ## → Привет, Мир!
ptl f"2 + 2 = {2 + 2}"    ## → 2 + 2 = 4
```

## Ввод

```
inp x                   ## прочитать строку в x
inp "Введи имя: " x     ## прочитать с подсказкой
```

**Важно:** `inp` всегда возвращает строку. Для чисел нужна конвертация:
```
inp "Число: " raw
L n = toInt(raw)        ## безопасно: вернёт nil если не число
## или
L n = int(raw)          ## жёстко: выдаст ошибку если не число
```

Проверка перед конвертацией:
```
inp "Число: " raw
if canInt(raw) {
    L n = int(raw)
    ptl f"Ты ввёл: {n}"
} el {
    ptl "Это не число!"
}
```

## Условия

```
if x > 10 {
    ptl "больше 10"
} elif x == 10 {
    ptl "ровно 10"
} el {
    ptl "меньше 10"
}
```

Операторы сравнения: `==` `!=` `<` `>` `<=` `>=`
Логические: `&&` (и), `||` (или), `!` (не)

```

    ptl "от 1 до 99"
}
if !b {
    ptl "b ложно"
}
```

## Циклы

### for — lp
```
lp i in rng(5) {        ## i = 0,1,2,3,4
    ptl i
}

lp i in rng(1, 6) {     ## i = 1,2,3,4,5
    ptl i
}

lp i in rng(0, 10, 2) { ## i = 0,2,4,6,8  (шаг 2)
    ptl i
}

lp item in [10, 20, 30] {  ## по элементам списка
    ptl item
}
```

### while — wh
```
L x = 0
wh x < 5 {
    ptl x
    x = x + 1
}
```

### Управление циклом
```
brk    ## break   — выйти из цикла
cnt    ## continue — перейти к следующей итерации
```

## Функции

```
fn greet(name) {
    ptl f"Привет, {name}!"
}

fn add(a, b) {
    rt a + b          ## rt = return
}

greet("Алекс")
L result = add(3, 4)   ## result = 7
```

Рекурсия работает:
```
fn fact(n) {
    if n <= 1 { rt 1 }
    rt n * fact(n - 1)
}
ptl fact(10)    ## 3628800
```

## Списки

```
L nums = [3, 1, 4, 1, 5]

nums[0]           ## первый элемент = 3
nums[0] = 99      ## изменить элемент
len(nums)         ## длина = 5
push(nums, 9)     ## добавить в конец
pop(nums)         ## убрать последний, вернуть его
sort(nums)        ## отсортированный новый список
rev(nums)         ## перевёрнутый новый список
sum(nums)         ## сумма всех элементов
max(nums)         ## максимум
min(nums)         ## минимум
count(nums, 1)    ## сколько раз встречается 1
idx(nums, 4)      ## индекс элемента 4 (или -1)
uniq(nums)        ## убрать дубликаты
```

Перебор списка:
```
lp x in nums {
    ptl x
}
```

Функции высшего порядка:
```
fn double(x)  { rt x * 2 }
fn isEven(x)  { rt x % 2 == 0 }
fn add2(a, b) { rt a + b }

map(nums, double)       ## [6,2,8,2,10] — применить функцию к каждому
flt2(nums, isEven)      ## [4] — отфильтровать
red(nums, add2, 0)      ## сумма через reduce
```

## Пример — полноценный калькулятор

```
## ── Калькулятор MinLang ──

fn calc(a, op, b) {
    if op == "+" { rt a + b }
    elif op == "-" { rt a - b }
    elif op == "*" { rt a * b }
    elif op == "/" {
        if b == 0 { rt "Ошибка: деление на 0" }
        rt a / b
    }
    elif op == "%" { rt a % b }
    elif op == "^" { rt pow(a, b) }
    rt "Неизвестная операция"
}

ptl "=== Калькулятор MinLang ==="
ptl "Введите 'q' для выхода"
ptl ""

wh T {
    inp "Первое число: " rawA
    if rawA == "q" { brk }

    if !canFlt(rawA) {
        ptl "Ошибка: введите число"
        cnt
    }

    inp "Операция (+,-,*,/,%,^): " op
    inp "Второе число: " rawB

    if !canFlt(rawB) {
        ptl "Ошибка: введите число"
        cnt
    }

    L a = flt(rawA)
    L b = flt(rawB)
    L res = calc(a, op, b)
    ptl f"= {res}"
    ptl ""
}

ptl "До свидания!"
```
