# MinLang — История версий

MinLang — минималистичный язык программирования с расширением `.ll`.  
Этот файл отражает постепенное развитие проекта — от простейшего
скрипта «переменные + принт» до полноценного интерпретатора с коллекциями,
функциями высшего порядка и стандартной библиотекой.

---

## v0.1 — «Доказательство концепции»

**Файл:** `v0.1_minlang.py`

Первый рабочий интерпретатор. Цель — убедиться, что связка
«токенайзер → парсер → интерпретатор» вообще работает.

**Что умеет:**
- Объявление переменных: `L x = 42`
- Вывод значения: `pt x`
- Типы: целые числа (`INT`) и строки (`STRING`)
- Комментарии: `## это комментарий`

**Чего нет:** арифметики, условий, циклов, функций.

**Пример кода:**
```
L name = "world"
L n = 99
pt name
pt n
```

## Как запускать

```bash
# Запустить файл
python minlang.py program.ll

# Интерактивный REPL (только v0.5+)
python minlang.py
```
